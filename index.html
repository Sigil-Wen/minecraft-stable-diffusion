Minecraft Map Art Maker

Upload an image to create an Add-On (Bedrock edition 1.16) that builds the image in your world, to view it on a map!

How this app works

Minecraft Map Art Maker resizes, pixelates and reduces the colour palette of your uploaded image to one that is shown in Minecraft maps. It then analyses the final image to identify a suitable minecraft block and coordinates for each pixel so that if those blocks are placed at those locations in your minecraft world, a map of the area will display the final image ! The placement is automated by writing several '/setblock' commands in function files, that can be applied through a behaviour pack and executed using the '/function' command in-game.

  • This app is free and open source online software,
  • Runs in your browser - no need to download or install programs, or send data to other websites/servers,
  • Valid for Minecraft on all devices that can use behaviour packs - Win 10, Android, iOS, Console, etc
